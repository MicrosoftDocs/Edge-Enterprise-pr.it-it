# Mapping dei criteri di Microsoft Edge Legacy a Microsoft Edge

Questo articolo associa i criteri di gruppo di Microsoft Edge Legacy (versione 45 e precedente) e le impostazioni di Gestione di dispositivi mobili (MDM) ai criteri pertinenti di Microsoft Edge versione 80. Per i criteri di Google Chrome, vedere l'articolo [Mapping dei criteri di Google Chrome a Microsoft Edge](microsoft-edge-policy-map-chrome-to-newedge.md).

> [!NOTE]
> Il mapping disponibile di seguito è concepito per facilitare la distribuzione iniziale di Microsoft Edge versione 80. Per un elenco definitivo dei criteri più recenti, vedere:
> - [Informazioni di riferimento sui criteri del browser](microsoft-edge-policies.md)
> - [Informazioni di riferimento sui criteri dell'aggiornamento](microsoft-edge-update-policies.md)
