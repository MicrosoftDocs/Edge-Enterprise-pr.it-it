# <span data-ttu-id="fe5cf-103">Creare variabili di directory dei dati utente di Microsoft Edge</span><span class="sxs-lookup"><span data-stu-id="fe5cf-103">Create Microsoft Edge user data directory variables</span></span>

## <span data-ttu-id="fe5cf-106">Variabili di percorso</span><span class="sxs-lookup"><span data-stu-id="fe5cf-106">Path variables</span></span>

<span data-ttu-id="fe5cf-107">Criteri per la modifica dei percorsi di directory di dati, ad esempio per la configurazione delle variabili di supporto [UserDataDir](microsoft-edge-policies.md#userdatadir) o [DownloadDirectory](microsoft-edge-policies.md#downloaddirectory).</span><span class="sxs-lookup"><span data-stu-id="fe5cf-107">Policies for modifying data directory paths (For example, configuring the [UserDataDir](microsoft-edge-policies.md#userdatadir) or [DownloadDirectory](microsoft-edge-policies.md#downloaddirectory) support variables.</span></span> <span data-ttu-id="fe5cf-108">Quando si configurano questi criteri, puoi usare le variabili anziché i percorsi hardcoded.</span><span class="sxs-lookup"><span data-stu-id="fe5cf-108">When configuring these policies, you can use variables instead of hard-coded paths.</span></span> <span data-ttu-id="fe5cf-109">Ad esempio, puoi archiviare i dati del profilo nei dati applicazione locali dell'utente in Windows invece che nel percorso predefinito.</span><span class="sxs-lookup"><span data-stu-id="fe5cf-109">For example, to store your profile data under user local application data on Windows instead of the default location.</span></span> <span data-ttu-id="fe5cf-110">Imposta il criterio [UserDataDir](microsoft-edge-policies.md#userdatadir) su **${local_app_data}\Edge\Profile**.</span><span class="sxs-lookup"><span data-stu-id="fe5cf-110">Set the [UserDataDir](microsoft-edge-policies.md#userdatadir) policy to **${local_app_data}\Edge\Profile**.</span></span> <span data-ttu-id="fe5cf-111">Nella maggior parte delle installazioni Windows 10, il percorso si risolve in *C:\Utenti\\&lt;Utente-corrente&gt;\AppData\Local\Microsoft\Edge\Profile*.</span><span class="sxs-lookup"><span data-stu-id="fe5cf-111">On most Windows 10 installations, this path resolves to *C:\Users\\&lt;Current-user&gt;\AppData\Local\Microsoft\Edge\Profile*.</span></span>

><span data-ttu-id="fe5cf-112">Se il criterio non viene impostato, Microsoft Edge usa i percorsi di directory predefiniti.</span><span class="sxs-lookup"><span data-stu-id="fe5cf-112">If left unset, Microsoft Edge uses default directory paths.</span></span> <span data-ttu-id="fe5cf-113">Un utente può usare il flag `--user-data-dir` della riga di comando per ignorare il valore predefinito.</span><span class="sxs-lookup"><span data-stu-id="fe5cf-113">A user can use the `--user-data-dir` command-line flag to override the default.</span></span>

### <span data-ttu-id="fe5cf-116">Linee guida per l'uso di variabili di percorso</span><span class="sxs-lookup"><span data-stu-id="fe5cf-116">Guidance for using path variables</span></span>

<span data-ttu-id="fe5cf-117">Prima di usare le variabili per i percorsi, vedi le linee guida seguenti.</span><span class="sxs-lookup"><span data-stu-id="fe5cf-117">Review the following guidance before using variables for paths.</span></span>
